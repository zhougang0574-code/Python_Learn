01

环境 & 基础语法

1.1 环境安装

推荐使用虚拟环境隔离项目依赖，等同于 Java 的 Maven/Gradle 作用域隔离。

PowerShell / Terminal 
# 1. 安装 Python（官网 python.org 下载 3.11+）
python --version          # 验证安装

# 2. 创建虚拟环境（类似 Java 项目的隔离 classpath）
python -m venv .venv

# 3. 激活虚拟环境
.venv\Scripts\activate    # Windows
source .venv/bin/activate # macOS / Linux

# 4. 安装依赖（类似 Maven pom.xml 中 dependency）
pip install langchain openai

# 5. 导出依赖清单（类似 pom.xml）
pip freeze > requirements.txt
pip install -r requirements.txt  # 恢复依赖

1.2 Python vs Java 核心差异

特性 Java Python
类型系统 静态强类型，编译期检查 动态类型，运行期检查（支持类型注解）
语句结束 分号 ; 换行（无分号）
代码块 花括号 { } 缩进（4 个空格，强制）
访问修饰符 public / private / protected 约定：_private / __name_mangling
主入口 public static void main(String[] args) if __name__ == "__main__":
null 值 null None
字符串 "hello"（双引号） "hello" 或 'hello' 均可
布尔值 true / false True / False（首字母大写）
打印输出 System.out.println() print()

1.3 变量 & 基本输出

Python 变量无需声明类型，赋值即创建。

☕ Java
String name = "Alice";
int age = 28;
double score = 99.5;
boolean active = true;

System.out.println(
  "姓名: " + name +
  ", 年龄: " + age
);
🐍 Python
name = "Alice"
age = 28
score = 99.5
active = True

# f-string（推荐，类似 Java 15+ 的文本块）
print(f"姓名: {name}, 年龄: {age}")
💡 Python 特色 f-string 是 Python 3.6+ 的字符串插值语法，性能好、可读性强，优先使用它而非 + 拼接或 .format()。

1.4 注释 & 文档字符串

Python
# 单行注释（用 # 号）

def add(a, b):
    """
    多行文档字符串（docstring）
    等同于 Java 的 Javadoc。

    Args:
        a: 第一个加数
        b: 第二个加数
    Returns:
        两数之和
    """
    return a + b

Python

☕ Java ArrayList
List<String> list =

list.add("apple");
list.add("banana");
list.get(0);        // "apple"
list.size();        // 2

🐍 Python list
lst = ["apple", "banana"]

# 可混合类型（Java 需要泛型Object）

nums = [3, 1, 4, 1, 5, 9]

nums.sort()                 # 原地排序 [1,1,3,4,5,9]
sorted(nums, reverse=True) # 返回新列表，不修改原列表

nums.extend([2, 6])        # 批量追加（类似 addAll）

2.4 元组（tuple）— 不可变列表

Python
# 元组不可修改，通常用于多返回值或固定数据

user = {"name": "Alice", "age": 28}

extra = {"city": "Beijing"}

passed = {k: v for k, v in scores.items() if v >= 80}

控制流

3.1 条件判断

☕ Java
if (score >= 90) {
    System.out.println("A");
} else if (score >= 60) {
    System.out.println("B");
} else {
    System.out.println("C");
}
🐍 Python
if score >= 90:
    print("A")
elif score >= 60:       # elif 不是 else if
    print("B")
else:
    print("C")
Python — 条件表达式（三元运算符）
# Java: String result = score >= 60 ? "pass" : "fail";
result = "pass" if score >= 60 else "fail"

# 逻辑运算符用英文（不是 &&, ||, !）
if age >= 18 and active:   # Java: age >= 18 && active
    print("成年且活跃")

if not active:             # Java: !active
    print("未激活")
⚠️ 常见坑 Python 中 0, "", [], {}, None 都是 falsy（假值），可以直接 if lst: 判断列表是否非空，而不用 if len(lst) > 0:。

3.2 for 循环

Python
# 遍历列表（类似 Java for-each）
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# range 生成数字序列（类似 Java for(int i=0; i<5; i++)）
for i in range(5):       # 0 1 2 3 4
    print(i)

for i in range(1, 6):    # 1 2 3 4 5
    print(i)

for i in range(0, 10, 2): # 0 2 4 6 8（步长2）
    print(i)

# enumerate：同时获取索引和值
for idx, fruit in enumerate(fruits):
    print(f"{idx}: {fruit}")

# zip：同时遍历多个列表
names = ["Alice", "Bob"]
ages  = [28, 32]
for name, age in zip(names, ages):
    print(f"{name} is {age}")

3.3 while 循环

Python
n = 0
while n < 5:
    print(n)
    n += 1            # Python 没有 n++ 语法！

# for/while 的 else 子句（Java 无此特性）
for i in range(5):
    if i == 3:
        break
else:
    print("循环正常结束（未触发break）")
☕ Java 开发者注意 Python 没有 i++ / i--，请使用 i += 1 / i -= 1。也没有 switch（Python 3.10+ 有 match 语句替代）。

3.4 match 语句（Python 3.10+，类似 switch）

☕ Java switch
switch (status) {
  case 200: System.out.println("OK"); break;
  case 404: System.out.println("Not Found"); break;
  default:  System.out.println("Other");
}
🐍 Python match
match status:
    case 200:
        print("OK")
    case 404:
        print("Not Found")
    case _:           # default
        print("Other")

4.2 参数类型

Python — 四种参数方式
# 1. 默认参数（Java 通过方法重载实现）
def connect(host, port=3306, timeout=30):
    print(f"连接 {host}:{port}, 超时 {timeout}s")

connect("localhost")               # 用默认值
connect("localhost", timeout=60)   # 关键字参数，顺序无关

# 2. *args：可变位置参数（类似 Java varargs）
def sum_all(*nums):                # nums 是一个 tuple
    return sum(nums)

sum_all(1, 2, 3, 4)              # 10

# 3. **kwargs：可变关键字参数（传字典）
def create_user(**info):           # info 是一个 dict
    print(info)

create_user(name="Alice", age=28, city="BJ")
# {"name": "Alice", "age": 28, "city": "BJ"}

# 4. 综合使用（LangChain 内部大量使用）
def api_call(endpoint, *args, method="GET", **headers):
    print(endpoint, args, method, headers)

4.3 Lambda（匿名函数）

☕ Java Lambda
// Java 8+ lambda
Comparator<String> comp =
  (a, b) -> a.length() - b.length();

List<String> words = ...;
words.sort(comp);
🐍 Python lambda
# lambda 只能写单个表达式
comp = lambda a, b: len(a) - len(b)

words = ["banana", "fig", "apple"]
words.sort(key=lambda x: len(x))
# ["fig", "apple", "banana"]

4.4 装饰器（Decorator）

装饰器是 Python 的核心特性，等同于 Java 的 AOP（面向切面编程），可以在不修改函数的情况下增强其功能。

Python
import time
from functools import wraps

def timer(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        elapsed = time.time() - start
        print(f"{func.__name__} 耗时: {elapsed:.3f}s")
        return result
    return wrapper

@timer                           # 语法糖，等价于 heavy = timer(heavy)
def heavy_task():
    time.sleep(1)
    return "done"

heavy_task()  # 自动打印: heavy_task 耗时: 1.001s
📌 LangChain 关联 LangChain 中的 @tool、@chain 装饰器就是用这个机制实现的，理解装饰器对学习 LangChain 非常重要。

4.5 内置高阶函数

Python
nums = [1, 2, 3, 4, 5]

# map：对每个元素应用函数（类似 Java Stream.map）
doubled = list(map(lambda x: x * 2, nums))
# [2, 4, 6, 8, 10]

# filter：过滤（类似 Java Stream.filter）
evens = list(filter(lambda x: x % 2 == 0, nums))
# [2, 4]

# Python 风格：更推荐用列表推导式代替 map/filter
doubled = [x * 2 for x in nums]
evens   = [x for x in nums if x % 2 == 0]
