# Python_Learn — 项目说明

## 项目目的

Java 后端开发者学习 **Python 基础**。本项目只覆盖 Python 语言本身，不包含 LangChain / LangGraph（那部分在另一个项目里学）。

最后一章专门解决一个实际痛点：看第三方库（如 LangChain）源码时，经常看不懂方法签名、不知道参数该怎么传 —— 这是类型注解 + `*args/**kwargs` + `inspect` 的知识，放在本项目里作为收尾。

## 项目结构

```
Python_Learn/
├── notebooks/
│   └── 序号_主题/
│       ├── 知识点.ipynb    # 概念讲解 + 示例代码（每章必须完整）
│       └── 练习.ipynb      # 练习题 + 解答（已掌握的主题练习量少，新主题练习完整）
├── .venv/                 # 虚拟环境（不提交 git）
└── PROJECT.md             # 本文件：进度 & 学习计划
```
